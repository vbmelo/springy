# Springy
Springy is a fullstack e-commerce application with strong focus in testing and QA.

# Tech Stack
### - Frontend
  - **React.Js**
### - Backend
  - **Java Spring Boot**
### - Database
  - **MongoDB**
### - Testing Tools
  - **JUnit, Mockito, Selenium**
### - Containers and Virtualization
  - **Docker**
